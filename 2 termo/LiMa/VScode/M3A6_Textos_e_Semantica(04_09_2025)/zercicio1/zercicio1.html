O que é HTML?

Linguagem de marcação:
HTML usa tags (etiquetas) para envolver diferentes partes do conteúdo, indicando como cada elemento deve ser exibido no navegador.

Estrutura da página:
O HTML define a organização da página, como cabeçalhos, parágrafos, listas, tabelas, links, etc.

Não é programação:
HTML não executa lógica de programação, como variáveis, loops ou funções.

Versão atual:
A versão mais recente é o HTML5, que inclui novas funcionalidades e elementos para a web moderna.
