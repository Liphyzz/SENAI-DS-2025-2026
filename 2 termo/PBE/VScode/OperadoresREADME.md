# OPERADORES PYTHON #


# --Quais são todos os operadores existentes em python?--

# R: Os operadores existentes em Python são:
# - Aritméticos: +, -, *, /, //, %, **
# - Relacionais: ==, !=, >, <, >=, <=
# - Lógicos: and, or, not
# - Bitwise: &, |, ^, ~, <<, >>, >>=, <<=
# - de atribuição: =, +=, -=, *=, /=, //=, %=, **=, &=, |=, ^=, >>=, <<=
# - de identidade: is, is not
# - de associação: in, not in


# --Explique cada operador (exemplo: + é usado para somar dois valores)--

# - Aritméticos:
#   + : usado para somar dois valores
#   - : usado para subtrair dois valores
#   * : usado para multiplicar dois valores
#   / : usado para dividir dois valores
#   // : usado para divisão inteira
#   % : usado para obter o resto da divisão
#   ** : usado para exponenciação

# - Relacionais:
#   == : verifica se dois valores são iguais
#   != : verifica se dois valores são diferentes
#   > : verifica se o valor da esquerda é maior que o da direita
#   < : verifica se o valor da esquerda é menor que o da direita
#   >= : verifica se o valor da esquerda é maior ou igual ao da direita
#   <= : verifica se o valor da esquerda é menor ou igual ao da direita

# - Lógicos:
#   and : retorna True se ambos os operandos forem True
#   or : retorna True se pelo menos um dos operandos for True
#   not : inverte o valor do operando

# - Bitwise:
#   & : realiza a operação AND bit a bit
#   | : realiza a operação OR bit a bit
#   ^ : realiza a operação XOR bit a bit
#   ~ : inverte os bits
#   << : desloca os bits para a esquerda
#   >> : desloca os bits para a direita
#   >>= : desloca os bits para a direita e atribui o resultado
#   <<= : desloca os bits para a esquerda e atribui o resultado

# - de atribuição:
#   = : atribui um valor a uma variável
#   += : adiciona um valor a uma variável e atribui o resultado
#   -= : subtrai um valor de uma variável e atribui o resultado
#   *= : multiplica uma variável por um valor e atribui o resultado
#   /= : divide uma variável por um valor e atribui o resultado
#   //= : realiza a divisão inteira de uma variável por um valor e atribui o resultado
#   %= : obtém o resto da divisão de uma variável por um valor e atribui o resultado
#   **= : realiza a exponenciação de uma variável por um valor e atribui o resultado
#   &= : realiza a operação AND bit a bit de uma variável por um valor e atribui o resultado
#   |= : realiza a operação OR bit a bit de uma variável por um valor e atribui o resultado
#   ^= : realiza a operação XOR bit a bit de uma variável por um valor e atribui o resultado
#   >>= : desloca os bits de uma variável para a direita e atribui o resultado
#   <<= : desloca os bits de uma variável para a esquerda e atribui o resultado

# - de identidade:
#   is : verifica se dois objetos são o mesmo objeto na memória
#   is not : verifica se dois objetos não são o mesmo objeto na memória

# - de associação:
#   in : verifica se um valor está presente em uma sequência (como listas, tuplas, strings)
#   not in : verifica se um valor não está presente em uma sequência (como listas, tuplas, strings)
